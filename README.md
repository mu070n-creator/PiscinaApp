# PiscinaApp

Projeto React (Vite) pronto para rodar.

## Como usar

1. Instale dependências:
```
npm install
```

2. Rode em modo desenvolvimento:
```
npm run dev
```

3. Faça build para produção:
```
npm run build
```

## Deploy

- Suba o repositório para o GitHub e importe no Vercel (Import Git Repository).
- Vercel detecta o projeto Vite automaticamente e faz o deploy.

## Notas

- Os dados (participantes e lista de compras) são salvos no `localStorage` do navegador.
- Senha organizador: `OsMelhores`
- Senha mestra para adicionar organizadores (apenas Murilo): `Muliru`
